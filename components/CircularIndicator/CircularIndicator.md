# CircularIndicator
Display progress by animating an indicator along an invisible circular track in a clockwise direction.

# Props

| Name             	| Type   	| Default  	|
|------------------	|--------	|----------	|
| size             	| Enum   	| small    	|
| animating        	| Bool   	|          	|
| hidesWhenStopped 	| Bool   	|          	|
| color            	| String 	|          	|


Note - Size can be either 'small' or 'large'
