# Icon

Supports the following icon libraries

1. Ionicons,
2. AntDesign,
3. Entypo,
4. EvilIcons,
5. Feather,
6. FontAwesome,
7. FontAwesome5,
8. Fontisto,
9. Foundation,
10. MaterialCommunityIcons,
11. MaterialIcons,
12. Octicons,
13. SimpleLineIcons,
14. Zocial,
