# TextButton

TextButton are used to create text based buttons.

## Props

| Name            | Type     | Default               |
|-----------------|----------|-----------------------|
| onPress         | Function | -                     |
| fontSize        | Number   | Theme.colors.fontSize |
| backgroundColor | String   | transparent           |
| color           | String   | Theme.colors.primary  |
| fontWeight      | String   | bold                  |
| title           | String   | -                     |
